Platillos

En el mantenimiento de Platillos encontramos los recursos necesarios para que los usuarios del sistema hotel san Carlos tengan el control de los platillos de manera práctica y sencilla. El mantenimiento cuenta con los siguientes campos:
- Código Platillo: Este será un id asignado al Platillo.
- Menú: Aquí se almacenará el id del menú relacionado al platillo.
- Nombre: Aquí se almacenará el nombre del platillo.
- Precio: Aquí se almacenará el precio asociado al platillo.
- Estado: este campo determinará la disponibilidad del campo.


Menu

En el mantenimiento de Menú del sistema del hotel san Carlos nos permite manipular los datos relacionados a la tabla Menú. Esto tiene como objetivo tener el control de los distintos menús del hotel y asi en el sistema evitar la redundancia de datos.
