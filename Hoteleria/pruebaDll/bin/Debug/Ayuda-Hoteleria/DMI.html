DMI

En el MDI encontramos las opciones principales con las que cuenta el sistema. Cada opción no permitirá realizar una serie de actividades las cuales tienen como objetivo hacer que el sistema hotel san Carlos sea facil de utilizar.


Herraminetas

La opción de heramientas nos permite acceder a las diferentes áreas que nos ofrece el navegador para realizar diversas tareas de manera práctica y sencilla.

Mantenimientos

En la opción de Mantenimientos encontraremos los distintos formularios que nos permitirán administrar el hotel ya que en el área de mantenimiento se encuentra el funcionamiento principal del sistema hotel san Carlos. Cada mantenimiento tendrá las opciones principales insertar, guardad, modificar, etc.

Ayuda

La opción de ayuda tiene la finalidad de dar a conocer el funcionamiento correcto del sistema y estar como soporte para las personas. En la opción ayuda del MDI cuenta con 2 opciones la que tiene el nombre MDI tiene el soporte de ayuda para el funcionamiento correcto del MDI y la opción mantenimiento ayuda tiene la opción de controlar el almacenamiento del control de las ayudas las cuales solo ciertos usuarios con conocimiento necesario tendrán acceso.
