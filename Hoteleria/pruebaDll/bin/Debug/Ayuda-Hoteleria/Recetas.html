Recetas

En el mantenimiento de recetas encontramos los recursos necesarios para que los usuarios del sistema hotel san Carlos tengan el control de las Recetas de manera práctica y sencilla. El mantenimiento cuenta con los siguientes campos:
- Código de Receta: Este será un id asignado a la receta.
- Receta: Aquí se almacenará el nombre o título de la receta.
- Estado: este campo determinará la disponibilidad del campo.
