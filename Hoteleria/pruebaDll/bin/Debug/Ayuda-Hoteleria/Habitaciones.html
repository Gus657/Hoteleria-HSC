Habitaciones

En el mantenimiento de habitaciones encontramos las opciones principales las cuales tienen como objetivo el control de las habitaciones para que el usuario pueda tener control en el sistema hotel san Carlos para poder almacenar las habitaciones y tener control del movimiento de los huéspedes.


Tipo de Habitación

Es una sub haría del control de habitaciones el cual tiene como objetivo registrar solo una vez los diferentes tipos de habitaciones con las cuales cuenta el hotel san Carlos.

Categoria de Habitación

Es una sub haría del control de habitaciones el cual tiene como objetivo registrar en el sistema los tipo de categoría de habitaciones esto tiene el objetivo al momento de relacionar habitaciones con tipo de habitaciones no exista redundancia en los datos insertados.
