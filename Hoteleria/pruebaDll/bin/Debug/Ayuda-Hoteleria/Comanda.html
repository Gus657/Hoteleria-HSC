Comandas

El mantenimiento de comandas nos permite registrar los servicios prestados dentro del Hotel San Carlos. Estos se realizan mediante el almacenamiento de un código y seleccionando el servicio utilizado para que a los usuarios que utilizan el sistema puedan almacenar registros de manera práctica y sencilla.

Servicios

El mantenimiento de servicios que presta el Hotel San Carlos por área registrada. Este mantenimiento tiene como objetivo tener el control de los servicios que presta el hotel a sus clientes. Los usuarios del hotel le asignan un código al servicio luego seleccionara el tipo de área en la cual se va a prestar el servicio y por último asignamos un nombre.
