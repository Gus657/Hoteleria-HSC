Objetos perdidos

El mantenimiento de objetos perdidos tiene como objetivo almacenar en el sistema del hotel san Carlos los distintos objetos perdidos ya sea en las habitaciones como en el hotel. El objetivo de este mantenimiento es ofrecer un mejor control para que al momento que el cliente pregunte al hotel se le pueda regresar el objeto que había dejado olvidado. El mantenimiento de objetos perdidos cuenta con los siguientes campos para que el usuario que utilice el sistema lo pueda registrar fácilmente.
- Código objeto: Este será un id asignado al objeto.
- Habitación: Aquí se almacenará el id de la habitación asociada.
- Fecha: aquí se almacenará la fecha en la cual se encontró el objeto.
- Descripción: Aquí se detallará las características del objeto.
- Estado: este campo determinará la disponibilidad del campo
